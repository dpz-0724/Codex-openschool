# 学习笔记目录（审计版）

这一页只放已经整理成公开 MD、读者打开就能获得方法或框架的笔记。不再放只有标题、没有正文的占位内容。

## 收录标准

一篇学习笔记必须同时满足：

1. 仓库里有可直接阅读的 `.md` 正文。
2. 内容有明确问题意识、方法框架或操作流程，不只是标题清单。
3. 已经脱敏：不包含账号后台、真实聊天、客户材料、个人路径、密钥、非公开数据或采集细节。
4. 能说明适合谁读、读完能拿走什么。
