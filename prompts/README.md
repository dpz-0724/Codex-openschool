# 第一期开源提示词清单

这里放适合教学和复用的提示词。第一期整理为 20 个模板，覆盖教学、代码审查、调试、产品需求、研究总结、Agent 工作流、视觉生成、PPT、视频脚本和学习复盘。

## 入口

- [第一期 20 个提示词模板](prompt-templates-20.md)
- [Codex 自我进化提示词](codex-self-evolution.md)
- [AI 教程视觉提示词方法：参考宝玉 skills](visual-prompts-baoyu-skills.md)

## 20 个模板目录

| # | 分类 | 模板 | 用途 |
|---:|---|---|---|
| 1 | 教学 | 苏格拉底式分步辅导模板 | 先追问、再提示、最后总结 |
| 2 | 教学 | 教师备课与差异化练习模板 | 课程目标、分层练习、课堂活动 |
| 3 | 代码审查 | PR 代码审查清单模板 | 找 bug、风险、测试缺口 |
| 4 | 代码审查 | 变更风险评估模板 | 评估发布风险和回归点 |
| 5 | 调试 | CI 失败诊断模板 | 读日志、列假设、给修复路径 |
| 6 | 调试 | 最小复现与根因分析模板 | 从现象走到可验证根因 |
| 7 | 产品 | PRD 一页纸模板 | 把想法变成可执行需求 |
| 8 | 产品 | Bug / Feature Issue 模板 | 生成清晰 issue |
| 9 | 研究 | 来源驱动研究简报模板 | 用证据写结论 |
| 10 | 研究 | 公开 Prompt 库二次改写模板 | 学结构，不复制原 prompt |
| 11 | Agent | 简单工作流 vs 自主 Agent 判断模板 | 判断该用 prompt、workflow 还是 agent |
| 12 | Agent | Codex 自我进化复盘模板 | 把重复任务沉淀成 skill |
| 13 | 视觉 | AI 教程视觉概念模板 | 教程封面、知识卡片、概念图 |
| 14 | 视觉 | 生产级图像提示词模板 | 主体、场景、构图、光线、禁用项 |
| 15 | PPT | 从文档生成演示稿模板 | 从资料到 PPT 大纲 |
| 16 | PPT | 高层汇报叙事模板 | 背景、洞察、决策、风险、下一步 |
| 17 | 视频 | 教程短视频脚本模板 | 钩子、演示、画面、总结 |
| 18 | 视频 | AI 视频镜头提示词模板 | 镜头、相机、动作、节奏 |
| 19 | 学习复盘 | Gibbs 六步复盘模板 | 事实、感受、评价、分析、结论、行动 |
| 20 | 学习复盘 | Cornell + 检索练习模板 | 线索问题、摘要、回忆测试 |

## 来源与改写原则

这些模板参考了 OpenAI、Anthropic、Google、GitHub、Microsoft、Atlassian、GitLab 等公开资料的方法论，但模板文本为本仓库重新组织的通用版本。发布时只保留来源链接和结构启发，不复制第三方长文本，不放本地私有上下文。
