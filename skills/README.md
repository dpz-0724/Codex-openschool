# 第一期开源 Skills 推荐清单

这份清单不是把本地 skills 源码直接公开，而是从教学和自用效率角度整理一批值得关注的 skills。每个条目都标注了用途、适合场景、来源信息和公开风险。

## 自动化浏览与网页操作

| Skill | 做什么 | 适合谁 | 作者 / 来源 | 来源状态 | 风险提示 |
|---|---|---|---|---|---|
| `web-access` | 统一处理联网搜索、网页抓取、登录态页面、动态渲染页面和浏览器自动化操作 | 需要让 Agent 查资料、读网页、操作网页后台的人 | 作者：一泽Eze；GitHub：`https://github.com/eze-is/web-access`；MIT | 外部公开项目 | 涉及登录态网页时要注意账号安全、站点规则和自动化检测 |
| `playwright` | 用真实浏览器做导航、表单填写、截图、数据提取和 UI 流程验证 | 想让 Agent 稳定操作网页和调试前端流程的人 | 基于 Playwright 生态，本地 skill 未见明确作者字段 | 本地封装 / 开源工具链 | 可能操作登录态页面，发布示例时不要带真实账号和业务数据 |
| `playwright-interactive` | 通过持久浏览器会话做更快的 UI 调试和交互式检查 | 需要反复调试网页、Electron 或本地 Web UI 的人 | 基于 Playwright 生态，本地 skill 未见明确作者字段 | 本地封装 / 开源工具链 | 不建议公开真实页面截图和内部系统路径 |
